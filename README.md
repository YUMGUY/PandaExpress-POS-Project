# PandaExpress-POS-Project
BEFORE RUNNING THE FIRST TIME (Make sure you are running NodeJS 16.14.x!):

1. cd to directory (here is PandaExpress-POS-Project)
2. npm i
3. cd client
4. npm i
5. npm run build
6. cd ../
7. npm run build


TO RUN PROGRAM:

- macOS: npm run dev
- Windows: npm run dev-windows
